# 🔐 Sistema de Login em Python

Projeto simples de autenticação via terminal desenvolvido em Python.

## Funcionalidades

- Login com usuário e senha
- Limite de 3 tentativas
- Bloqueio automático após erro
- Feedback ao usuário

## Tecnologias
- Python

## Aprendizados

- Estruturas condicionais
- Laços de repetição (while)
- Controle de estado
- Validação de entrada
